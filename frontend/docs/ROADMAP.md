# BreakPoint Development Roadmap

## Project Overview

BreakPoint is a modern coding practice platform where users intentionally debug broken code, solve programming challenges, and receive AI-powered feedback.

Development follows a milestone-based workflow. Only one milestone should be worked on at a time.

Legend:
- ⏳ Planned
- 🚧 In Progress
- ✅ Completed
